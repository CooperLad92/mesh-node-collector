# Mesh Node Collector

A quick-start React + Vite prototype for a Meshtastic-style collectible node card app.

## 1. Install dependencies

```bash
npm install
```

## 2. Run locally

```bash
npm run dev
```

## 3. Build for production

```bash
npm run build
```

## 4. Deploy to GitHub Pages

First edit `package.json` and replace:

```json
"homepage": "https://YOUR_GITHUB_USERNAME.github.io/mesh-node-collector"
```

with your real GitHub username.

Then run:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/mesh-node-collector.git
git push -u origin main
npm run deploy
```

Then in GitHub:
- open the repo
- go to **Settings**
- go to **Pages**
- confirm it is serving from the published branch

## Phone-only option

You can edit and upload this from Android using:
- **Replit** in Chrome
- **StackBlitz** in Chrome
- **GitHub mobile + a web IDE**

But a laptop is still much easier for the first deploy.
